# **App Name**: LibriGuard

## Core Features:

- Secure Student Check-in: Allows library staff to check students into the library by entering their unique student ID, accurately logging their entry time.
- Live Library Occupancy View: Displays a real-time list of all students currently checked into the library, providing an instant overview of current occupancy.
- Student Profile Lookup: Enables staff to view essential student profile details, including their current check-in/out status, upon successful ID entry.
- Effortless Student Check-out: Provides a streamlined process for staff to mark a student as having exited the library, recording their departure time.
- Staff Login & Access: A secure authentication system ensuring only authorized library staff can access and manage student entry and exit data.
- AI-powered Library Flow Insights Tool: An AI tool that analyzes current and historical occupancy data to provide staff with insights into peak usage times and common student flow patterns.

## Style Guidelines:

- Primary color: A calm, muted green (#519763) evoking nature, tranquility, and academic growth.
- Background color: A very light, desaturated green (#EEF5EF) providing a clean and spacious canvas, subtly harmonizing with the primary hue.
- Accent color: A fresh, bright yellowish-green (#A1E070) to highlight interactive elements and important statuses, creating a pleasant contrast.
- Headline and body text font: 'Inter' (sans-serif), chosen for its modern, highly readable, and neutral aesthetic, suitable for an administrative interface.
- Use clear, concise, and friendly line icons for actions like check-in, check-out, and profile viewing to maintain a welcoming yet efficient atmosphere.
- Adopt a clean, intuitive layout with ample whitespace, prioritizing readability and quick data access for library staff.
- Implement subtle transition animations for status updates and successful actions (e.g., student checked in), providing gentle visual feedback without being distracting.
